Como a ideia desse projeto é ser um framework, aqui está o status da implementação dos módulos core:

- [x] Estrutura para módulos de autenticação, autorização, criptografia, hash, rate limit, etc.
- [x] Estrutura para configuração de mensageria usando RabbitMQ.
- [x] Estrutura para Webhooks e Websockets.
- [x] Suporte a internacionalização (i18n).
- [x] Suporte para comandos via CLI para instalação do projeto e CRUD Maker.
